A Live ISO tartalma

aptosid only provides Debian dfsg-free software on the live-ISO, a nem-ingyenes firmware-kkel kapcsolatos információkért kérlek látogass el a linkre

Az ISO teljes mértékben a Debian Sid-ből készült (2010-06-13 dátummal) kiegészítve és stabilizálva a aptosid saját csomagjaival és script-jeivel. It comes with the latest enhanced aptosid kernel, which is based on the most recent vanilla kernel 2.6.3X.xx. Az ACPI és DMA alapértelmezetten engedélyezett.

A complete manifest of each of the release variants is included on each download mirror: aptosid Mirrors, Downloading and Burning.

Release ISO variations

aptosid offers 5 up-to-date live-ISO entry point release variations to Debian Sid and installing does not usually take longer than 10-20 minutes.

The variations are:
  1. KDE lite 32 bit , live-ISO approximately 600MB:
    contains a bare minimum set of applications for the KDE desktop. Adding more applications to suit your long term requirements is easily done via apt.
  2. KDE lite 64 bit , live-ISO approximately 600MB:
    contains a bare minimum set of applications for the KDE desktop. Adding more applications to suit your long term requirements is easily done via apt.
  3. XFCE 32 bit, live-ISO approximately 500MB:
    contains a fully featured Desktop Environment (not minimal), and has all the applications you need to be productive immediately. It has a smaller footprint than KDE.
  4. XFCE 64 bit , live-ISO approximately 500MB:
    contains a fully featured Desktop Environment (not minimal), and has all the applications you need to be productive immediately. It has a smaller footprint than KDE.
  5. KDE-full, a combined 32 bit + amd64 bit, live-ISO approximately 2.5GB:
    contains a full suite of KDE applications with 16 languages supported and includes amongst other things, office productivity suites like LibreOfficeg and web browsers like firefox

  • Optikai lemezmeghajtó vagy USB média.

    • any x86-64/ EM64T -képes vagy újabb processzor
    • teljes i686 parancskészlettel.
  • RAM requirements:
    • KDE: ≥512 MB RAM (≥1 GB RAM recommended), ≥1 GB RAM for liveapt.
    • XFCE: ≥512 MB RAM.
  • VGA grafikus kártya, amely minimum 640x480 képpont felbontásra alkalmas.
  • Optikai lemezmeghajtó vagy USB média.
  • ≥3 GB hely a merevlemezen (ajánlott ≥10 GB).

Alkalmazások és Segédeszközök

The aptosid-ISO holds a variety of different programs and tools that cover almost any need of a desktop-user in everyday life.

Choice of 2 major cutting edge Desktop Managers to access programs/applications:
KDE (en + de), a top of the range desktop manager
Xfce4 4.6.1.x (en + de), a medium weight desktop manager.

Fluxbox (en + de) is included on both variations as a lightweight desktop manager option.

For surfing the Internet, depending on the release variation, there is Konqueror and firefox.

Office Productivity applications (again depending on the variation and to name only a few), range from Calligra Suite, LibreOffice, Abiword, Gnumeric through file managers like Dolphin and Thunar.

To configure Networking and Internet connections there is Ceni and for WIFI see our WIFI Roaming documentation.

Non-free driver information is here.

For partitioning a hard drive, we have provided cfdisk and GParted and gparted has an ntfs resize facility.

Diagnostic tools like Memtest86+, an Advanced Memory Diagnostic Tool, are provided.

All ISO variations contain a full set of command line interface tools. A complete manifest of each of the release variants is included on each download mirror: aptosid Mirrors, Downloading and Burning.

Jogi nyilatkozat

Ez egy kísérleti szoftver, mindenki saját felelősségére használja. A aptosid project fejlesztőit és csapat tagjait az esetleges hardver- vagy szoftversérülés, adatvesztés, vagy a használat során bekövetkezett közvetett károkért semmilyen felelősség nem terheli. Ha nem fogadod el ezeket a feltételeket, akkor a szoftver disztribúció felhasználásra nem vagy jogosult.
