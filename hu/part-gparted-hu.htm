Az alapok

A partition must have a filesystem. Linux knows different filesystems to use. Ext3 is the recommended format for aptosid. ext2 is handy as a storage format as an MS Windows™ driver is available for data-swapping. Ext2 Installable File System For MS Windows.

Mindennapi használatra az ext4 fájlrendszert ajánljuk, mivel ez a aptosid fő fájlrendszere, ezért jól karbantartott.

Using KDE Partition Manager & Gparted

Creating and managing partitions is not something that is typically done every day. Therefore, a good idea is to read this guide once, to get comfortable with the concepts and some of the panels that will appear. The following assumes that you are using a live-iso . Should you be hard disk installed <sudo> will not work, refer to About Xapps whilst in X as root .

KDE Partition Manager, if installed on the live-iso - run in a terminal:

sudo partitionmanager

Gparted, if installed on the live-so - run in a terminal:

sudo gparted
Whilst somewhat obscured within the menu choices, gparted will let you see GPT disks.

If you need to partition a USB drive, (or SD and SDXC cards), whilst hard disk installed you will need to adjust your commands to include root using Xapps. Refer to About Xapps whilst in X as root else use the commandline tools cfdisk or gdisk.

 

  • When GParted or KDE Partition Manager runs , a window is opened, and drives are scanned.

    In KDE Partition Manager the drives are shown in a list on the left.

    In Gparted, if you click the menu (at the top left), a pop down is presented. You can select to refresh the display of the drives on your system.

  • Szerkesztés

    A Szerkesztés (Edit) menüpont három, kiszürkített menüpontot tartalmaz, amelyek döntő fontosságúak lesznek:
    Az utolsó művelet visszavonása (Undo last operations)
    Műveletek alkalmazása (Apply)

  • Nézet

    • Merevlemez információ

      A merevlemez információ panelen láthatjuk a merevlemez típusát, méretét és egyéb hasznos információkat. Az itt látható információk segítségével győződhetünk meg arról, hogy valóban a kívánt merevlemezzel dolgozunk-e.

    • Sorbanálló műveletek

      Az alsó ablakban láthatjuk a sorban álló műveletek listáját. Ennek segítségével ellenőrizhetjük, hogy hány műveletet kell majd elvégezni.

  • Eszközök

    Az Eszközök menüben állítható be vagy változtatható meg a kötet címkéje, amennyiben nem megfelelő.

  • Partíció

    Ebben a menüben találjuk a legfontosabb funkciókat. Többféle művelet elvégzésére van lehetőség, és ezek között találunk veszélyeseket is.

    Törlés (Delete), egy adott partíció végleges törléséhez. A művelet elvégzéséhez előbb ki kell választani a megfelelő partíciót.

    Az Átméretezés/Mozgatás funkcióval partíciókat méretezhetünk át.

  • Új partíció létrehozása

    Egy még nem használt terület kiválasztását követően az eszköztáron az Új gomb megnyomásával lehet új partíciót létrehozni. A felbukkanó ablakban kiválasztható a mérete; hogy elsődleges, kiterjesztett vagy logikai partíció legyen; illetve a fájlrendszer típusa.

  • Ha hibát vétettél

    Hibás partíció létrehozás esetén használható a Törlés funkció a kiválasztott partíció törlésére, illetve ha még nem hagytuk jóvá az 'Alkalmaz' gomb megnyomásával, akkor használhatjuk a 'Mégsem' (Undo) gombot is.

  • Átméretezés/Mozgatás

    Egy partíció átméretezéséhez válasszuk az Átméretezés/Mozgatás gombot; ekkor egy új ablak jelenik meg. A partíció méretének csökkentéséhez vagy növeléséhez használható az egér és a nyíl billentyűk is.

    A beállítások elvégzése után kattints az Alkalmaz gombra, a változtatások csak azután hajtódnak végre.

    A feladat végrehajtásának időtartama a partíció méretétől függ.

    A partíciós tábla szerkesztését követően a számítógépet újra kell indítani, hogy a aptosid újra tudja olvasni a partíciós táblát.

NTFS partíciók átméretezése

Az NTFS partíció átméretezése a rendszer újraindítását igényli! NE TÉGY semmiféle egyéb változtatást ezen a partíción újraindítás előtt, másképp hibákba fogsz ütközni.

  • A gép újraindítása után a MS Windows™ elindulásakor egy speciális képernyő indul el, és egy üzenetet fogsz látni a lemez konzisztenciájával kapcsolatban: Fájlrendszer ellenőrzése a következőn C :
  • Hagyd az AUTOCHK -t lefutni: A fájlrendszer ellenőrzés kötelező az NT esetében egy átméretezés után.
  • Az ellenőrzés végeztével a számítógép ismét automatikusan újra fog indulni. Ez biztosítja a tökéletes működést.
  • Újraindítás után a Windows XP a megszokott módon indul, de meg kell várnod míg a rendszer végleg bebootol és betölti a login képernyőt!

Teljes GParted dokumentáció: A teljes körű dokumentáció az alábbi weboldalon található: GParted.

NTFS-partíciók írása ntfs-3g segítségével

Figyelem: Bár az ntfs-3g kernel modul stabilnak minősített, sose használd külső biztonsági mentés nélkül és természetesen ne éles rendszeren! Ha megteszed, a te hibád ha adatvesztés következik be, tehát csak saját felelősségedre használhatod!

Nyiss egy parancssort és add ki a következő parancsokat (Lásd A merevlemez partícionálása - A lemezek elnevezése fejezetet):

sux
apt-get update && apt-get install ntfs-3g
umount /media/xdxx
mount -t ntfs-3g /dev/disk/by-uuid/xxyyzz[etc] /media/xdxx
Kilépés a konzolból: exit

Az NTFS köteted fel lett csatolva rw és most már írni is fogod tudni. De még egyszer figyelmeztetünk! Csak vészhelyzetben használd, nem javasolt a napi szintű alkalmazása.

exfat

exfat is a proprietary file format covered by patents held by Microsoft ™ and commonly found on SDXC cards. To mount/read/write to the file system, simply install the exfat-fuse package:

apt-get update && apt-get install exfat-fuse
