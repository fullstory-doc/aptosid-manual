Live-ISO:ns innehåll

aptosid only provides Debian dfsg-free software on the live-ISO. Vänligen se denna länk för vidare information om ofria källor för firmware

ISO:n är helt och hållet baserad på Debian Sid (uppdaterad den 2009-07-xx) och är berikad samt stabiliserad med aptosid egna paket och script. It comes with the latest enhanced aptosid kernel, which is based on the most recent vanilla kernel 2.6.3X.xx. ACPI och DMA är aktiverade som standard.

Release ISO utgåvor

aptosid erbjuder 5 up-to-date live-ISO utgåvor av Debian Sid och installationen brukar inte ta längre tid än 10-20 minuter.

Utgåvorna är:
  1. KDE lite 32 bit , live-ISO ungefär 600MB: innehåller en minimal uppsättning program för KDE4 skrivbordet. Lägga till fler program till era långsiktiga krav är lätt gjort via apt.
  2. KDE lite 64 bit , live-ISO ungefär 600MB: innehåller en minimal uppsättning program för KDE4 skrivbordet. Lägga till fler program till era långsiktiga krav är lätt gjort via apt.
  3. XFCE 32 bit, live-ISO ungefär 500MB: innehåller en fullfjädrad Skrivbordsmiljö (ej minimal), och har alla program du behöver för att vara produktiv direkt. Den har ett mindre "footprint" än KDE4.
  4. XFCE 64 bit , live-ISO ungefär 500MB: innehåller en fullfjädrad Skrivbordsmiljö (ej minimal), och har alla program du behöver för att vara produktiv direkt. Den har ett mindre "footprint" än KDE4.
  5. KDE-full, kombinerad 32 bit + amd64 bit, live-ISO ungefär 2.5GB: innehåller en komplett uppsättning av KDE4 applikationer med 16 olika språk och innefattar bland annat kontorsprogramspaketet LibreOffice och webbläsaren firefox

Applikationer och verktyg

Lägsta systemkrav för KDE4 och Xfce

AMD64

  • CPU krav:
    • AMD64
    • Intel Core2
    • Intel Atom 330
    • alla x86-64/ EM64T kapabla CPU eller nyare
    • nyare 64 bit kapabel AMD Sempron och Intel Pentium 4 CPUs (Titta efter "LM" flagga i / proc / cpuinfo eller använda infobash-v3).
  • RAM krav:
    • KDE: ≥512 MB RAM (≥768 MB RAM rekommenderas), ≥1 GB RAM för liveapt.
    • XFCE: ≥512 MB RAM.
  • VGA kompatibelt grafikkort som klarar minst 640x480 pixel upplösning.
  • Optisk hårddisk eller USB-media
  • ≥3 GB HDD utrymme, ≥10+ GB rekommenderas.

i686

  • CPU krav:
    • Intel Pentium pro/ Pentium II
    • AMD K7 Athlon (not K5/ K6)
    • Intel Atom N-270/ 230
    • VIA C3-2 (Nehemiah, not C3 Samuel or Ezra)/ C7
    • alla x86-64/ EM64T kapabla CPU eller nyare
    • hela i686 kommandouppsättning krävs.
  • RAM krav:
    • KDE: ≥512 MB RAM (≥1 GB RAM rekommenderas), ≥1 GB RAM för liveapt.
    • XFCE: ≥512 MB RAM.
  • VGA kompatibelt grafikkort som klarar minst 640x480 pixel upplösning.
  • Optisk hårddisk eller USB-media.
  • ≥3 GB HDD utrymme, ≥10+ GB rekommenderas.

Program och verktyg

aptosid-ISO:n innehåller en mängd olika program och verktyg vilka täcker nästan samtliga behov en datoranvändare har i vardagen.

Choice of 2 major cutting edge Desktop Managers to access programs/applications:
KDE4 (en + de), En av skrivbordshanterarnas toppmodell
Xfce4 (en + de), en lite resurssnålare skrivbordshanterare

Fluxbox (en + de) följer med i båda utgåvorna som lättviktigt skrivbordshanteringsval.

Surfa på Internet, lite beroende på utgåva, finns Konqueror och firefox.

Kontorsprogramspaket (också beroende på utgåva och för att ge ett par exempel), Koffice, Openoffice.org, Abiword, Gnumeric och filhanterare som Dolphin och Thunar.

För att konfigurera Internet och nätverk finns Ceni och för WIFI läs våran WIFI Roaming dokumentation.

Ofria drivrutins information finns här.

För att partitionera en hårddisk, inkluderar vi cfdisk och GParted och gparted har möjlighet att storleksändra ntfs partitioner.

Diagnosverktyg som Memtest86+, och Advanced Memory Diagnostic Tool, är inkluderade.

Alla ISO utgåvorna innehåller en full uppsättning av kommandoradsgränssnitts verktyg. Ett komplett manifest för varje release utgåva finns med på varje nedladdnings spegel:: aptosid Nedladdningsspeglar, Nedladdning och Bränning.
