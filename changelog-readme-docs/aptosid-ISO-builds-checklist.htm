aptosid ISO Quality Assurance checklist

This is core QA checklist for things that we may tend to forget to check and should check, it is not meant be a definitive guide to check for corner cases.

Preset options:

  • md5sum
  • Boot line options (for example: F4 language F3 Timezone F5 Kernel default)
  • All desktop icons work as expected
  • Terminal tests (init 3 back to init 5 blah blah)
  • Installer reads the partitions properly
  • Partitioner works as expected
  • liveapt
  • memtest

Install tests:

  • Install to an image
  • Install to a stick with raw and install-usb-gui
  • Install to a HD using plain
  • Install to a HD using lvm
  • Install to a HD using crypt

Complex behaviours:

  • fromiso with persist with ext2 ext3 ext4 and fat
  • NTFS read/write

Other QA metrics:

  • wlan: especially if no firmware is needed
  • Samba
  • Printers
  • Enable system sounds [requires logout /login]
  • Play a media file ( audio/video)
  • bluetooth: at least device pairing

Art QA:

  • verify gfxboot-theme, kdm-theme, ksplash-theme, wallpaper, gdm-theme, & xfce-splash-theme
