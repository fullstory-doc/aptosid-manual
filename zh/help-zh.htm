何处获取帮助

获得及时的帮助可能就会是以眼泪告终与继续你人生中意义重大之举间的差异。该话题将给你一个渠道，去何处以及如何获取作为一款发行版的sidux所提供的帮助。话题分解如下。

  • 论坛及维基
  • IRC
  • Tools to know in init 3 and text mode (tty)
  • Using IRC in text mode and/or init 3
  • Web browsing in text mode and/or init 3
  • infobash

The aptosid forums

The aptosid forums offer a place to post questions, and receive answers to those questions, please do search the forums before you post, as others may have already asked the very same question. The The forum is in English.

The aptosid Wiki is free to use and edit by and for all aptosid users.

We hope for contributions from Linux users of all knowledge levels, since this wiki is intended to help users of all levels. The few minutes you donate to our wiki can save other users (and maybe yourself) in need of solutions, hours of trouble and searching. The Wiki.

IRC互动现场帮助

IRC使用协议

千万不要以‘root’根用户身份使用IRC .. 以普通用户的身份进入IRC聊天室，你是受欢迎的，但以root用户身份，决不：如果你有困难，请立即讲出来。

连到#sidux

有两种方法可以获取现场在线帮助
1）点击桌面上的sidux-irc图标，或者使用另一聊天客户端
2) 点击sidux.com菜单里的Meet the Team

Konversation

最简单的方法就是点击桌面上的sidux-irc 图标或者使用Kmenu因为konverstion已经预设好了。

如果你偏爱另一聊天客户端，需把服务器选项设置为：

irc.oftc.net
端口 6667

sidux.com网站上的irc

去sidux.com然后点击菜单列表里面的Meet the Team in the menu list. This will give you options to use a Chat (cgi) or Chat (java) web based irc chat client.

在昵称那块敲入你选定的昵称，信道那块敲入#sidux，然后点击‘登入’与／或跟随提示。

!paste

Debian张贴栏

在信道里!paste将是你的一个好帮手，不然你可能会被泛滥的消息给冲走。所以如果你被要求用!paste提供你的konsole输出，在你的聊天那块键入!paste（或者某人可能已经代劳，为你唤出了仿真陈述）。然后会返回这样的结果：

"paste" is
http://paste.debian.net/ :please choose no
syntax highlighting;make your paste;
then give #sidux the link to your paste

［译注：返回的消息即为仿真陈述（factoid），大意是“ ‘张贴栏’为http://paste.debian.net/：请选择无语法高亮显示；贴出你的帖子； 然后在#sidux给出你的帖子的链接”］

所有你要做的就是点击那个链接，它会带你去到那个网站，把你被要求提供的输出贴到张贴那块，点击发送。

然后你会看到你的记录，连同链接帖子至IRC信道的若干选项已经被加入了数据库。那正是信道那边要求你提供的链接。（实例：链接至你的记录用：http://paste.debian.net/524 ）。Debain张贴栏可以设置显示一特定时间段（最多72小时）以内的信息。也有选项让你删除帖子／数据。

sidux-paste

sidux-paste允许你贴出来自终端和tty的文件输出，因此如果你在init 3下遇到了困难，它会是一个理想的帮手。sidux-paste使用链接http://rafb.net，张贴的输出24小时内可见。

你可以以普通用户或root用户身份启用sidux张贴栏，但注意有些人要求的输出是需要root用户身份才能获取的。

sidux-paste <文件>
或
sidux-paste -c "命令1 --选项 参数; 命令2"
sidux-paste <文件>实例
aptosid-paste /etc/fstab
Your paste can be seen here: http://rafb.net/p/n8miQp85.html

http://rafb.net/p/n8miQp85.html是你需要在IRC #sidux给出的链接

sidux-paste -c实例
aptosid-paste -c "fdisk -l; cat /etc/fstab"
Your paste can be seen here:http://rafb.net/p/ZrhlVc59.html

http://rafb.net/p/ZrhlVc59.html是你需要在IRC #sidux给出的链接

在纯文本模式下需要了解的工具 (tty) and init 3

通常当你要进行dist-upgrade（或者你的系统遇到了非常糟糕的情况）你才会进入纯文本模式或init 3。

gpm

纯文本模式下一个很好的工具是gpm。它让你能在终端／konsole之间使用鼠标进行复制和粘贴。

gpm在sidux上已经预设好了，不然请照如下做：

gpm -t imps2 -m /dev/input/mice

确保/etc/gpm.conf存在

service gpm restart

现在你在纯文本模式（tty）下鼠标可用了

File browsing and text editing

Midnight Commander is a text-mode (tty) full-screen file manager and text editor installed by default and is not complex to use.

Apart from the normal keyboard commands it also responds to mouse clicks for navigation because of gpm.

mc brings up the file system while mcedit brings up a blank file or you can go directly to a file.

Example to go directly to a file:

mcedit /etc/apt/sources.list.d/debian.list

You can now edit the file to change parameters and save the alteration to have immediate effect.

Read:

man mc

在纯文本模式下访问irc #sidux的支持

IRC使用协议

千万不要以‘root’身份运行IRC .. 以普通用户的身份进入IRC聊天室，你是受欢迎的，但以root用户身份，决不：如果你有困难，请立即讲出来。

在纯文本模式与／或init 3下的irc

sidux默认的是irssi。

在init 3下时，切换至另一个终端／konsole，你可以通过：

# CTRL-ALT-F2
$ siduxbox 登入: <用户名> <密码>（不要用root用户）
然后键入
$ sidux-irc（这会起动irssi）

或者你你安装了另外的客户端，比如weechat：

# CTRL-ALT-F2
$ siduxbox 登入:  <用户名> <密码>（不要用root用户）
然后键入
$ weechat-curses

现在通过端口6667，连线到irc.oftc.net。连上之后，修改你的昵称：

/nick 选定的昵称

加入sidux信道：

/join #sidux

如果你想连线到另一个服务器：

/server 服务器名

如果信道有新的动态，你会在底部的条上看到数字出现，要切换至那个信道，ALT-1 ALT-2 ALT-3 ALT-4 等等 ...

要离开，

/exit

例如，你正在进行dist-upgrade，要查看进度

比如回到d-u
$ CTRL-ALT-F1
然后回到irc
# CTRL-ALT-F2

在纯文本模式与／或init 3下浏览网页

wm3与／或elinks让你能在终端／konsole或纯文本模式与／或init 3下浏览网页。

检查w3m或elinks已安装，不然请照如下做：

apt-get update
apt-get install w3m elinks

使用w3m的一个例子，打开一个终端／konsole：

$w3m URL
或
w3m ?
或
w3m sidux.com

如果你看到的是一整块黑屏，只有些说明出现在底部，不用恐慌，使用下面的组合键

SHIFT+U

然后你会看到它打开到了你的本地磁盘，像是Goto URL: file:///home/用户名/$，就一路backspace直到GOTO URL，键入，比如：

http://sidux.com

现在你可以浏览sidux.com或其他你选择的网站了。

要么在init 3下：

# CTRL-ALT-F2
$ siduxbox 登入:  <用户名> <密码>（不要用root用户）
然后键入
$ w3m sidux.com
要回到the d-u
$ CTRL-ALT-F1

直接去手册

$w3m manual.sidux.com

w3m主页

It is advisable to familiarise yourself with using elinks/w3m, irssi/weechat and midnight commander before an emergency happens, if one should occur. This would be a good page to print out as a reference on how to get online help in such emergencies.

Infobash

infobash是一个系统信息脚本，独立于某一特定的IRC（Internet Relay Chat，互联网接力聊天客户端。这个系统信息脚本能将你的硬件和软件的所有东西都显示给信道里的人看，以便他们帮助你诊断问题，...，或者你自己闲来无事时通过konsole惊叹一下你的系统明细和内核版本。

To use the infobash script in Konversation type into the chat box:

/cmd infobash -v3

To use the infobash script in other clients type into the chat box:

/exec -o infobash -v3
或
/infobash -v3

在konsole，键入：

infobash -v3

阅读关于infobash一切的全篇指导:

man infobash
