如何对你的HD进行分区（使用Gparted/KDE Partition Manager）

警告：关于磁盘命名请参考UUID，分区标记及fstab，因为sidux现在默认采用的是UUID
Partitioning tools may request a root password, type sux then your password. On a Live-ISO none is set sux then press 'enter' . See: Live Mode

基础

一个分区必须有一个文件系统。Linux知道使用不同的文件系统。有ReiserFS，Ext4，以及面向有经验的用户的XFS和JFS。Ext2作一存储格式很好，因为一个供数据交换的Windows驱动器是现成的。. Ext2 Installable File System For MS Windows.

Using KDE Partition Manager & Gparted

Creating and managing partitions is not something that is typically done every day. Therefore, a good idea is to read this guide once, to get comfortable with the concepts and some of the panels that will appear. The following assumes that you are using a live-iso . Should you be hard disk installed <sudo> will not work, refer to About Xapps whilst in X as root .

KDE Partition Manager, if installed on the live-iso - run in a terminal:

sudo partitionmanager

Gparted, if installed on the live-so - run in a terminal:

sudo gparted
Gparted will you let create and see GPT disks.

If you need to partition a USB drive, (or SD and SDXC cards), whilst hard disk installed you will need to be root. Refer to About Xapps whilst in X as root else use the commandline tools cfdisk or gdisk.

 

  • When GParted or KDE Partition Manager runs , a window is opened, and drives are scanned.

    In KDE Partition Manager the drives are shown in a list on the left.

    In Gparted, if you click the menu (at the top left), a pop down is presented. You can select to refresh the display of the drives on your system.

The following screenshots are of Gparted. KDE Partition Manager behaves in much the same manner.

  • 编辑（Edit）

    它会打开一个新的窗口，在其上你可以看到支持的文件系统及一些分区编辑选项。??It opens a new window from which you can see the supported file systems and some partition editing options.??

    位于菜单栏从左数的第二个位置上。它显示有两个泛灰的非常有用的功能：撤销，及应用，同时也让你可以访问／检视其他区域。??The menu is second from the left. It shows two greyed out functions which are quite useful: Undo, and Apply and also allows you to access/view other areas.??

  • 检视（View）

    • 设备信息

      硬盘信息条显示有硬盘的细节，比如模型，大小等。这个条在一个多硬盘的系统上很有用，那上面的信息可用以确定考察的确实是想要考察的硬盘。

    • 待定操作

      下面的窗口是一个待定操作的列表。这一信息也有用因为它明确地列出了一系列待定操作。

  • 设备（Device）

    设备菜单让你可以设定一个磁盘标签。如果你觉得当前的磁盘标签不合适，你可以利用这一选项来作修改。

  • 分区菜单

    这个菜单相当重要。它让你可以执行很多操作，包括一些危险操作。

    如果你想删除一个分区，选择删除（Delete）要执行删除操作，你必须先选定分区。

    放缩／移动（Resize/Move）是一个拥有的功能。

  • 创建一个新的分区

    如果你已经选择了一块未分配的区域，工具栏上的新建（New）按钮让你可以创建一个新的分区。一个新的窗口会出现，让你选择你想要的大小，是否将其作为预选（Primary），拓展（Extended）或逻辑（Logical）分区，以及文件系统。

  • 如果你失误了

    如果你失误了，你可以用那个删除（Delelte）按钮删除选定的分区，或者如果你还没有将你的决定付诸应用，你可以用那个撤消（Undo）按钮。

  • 放缩／移动（Resizing/Move）

    如果你想放缩你选定的分区，点击放缩／移动（Resize/Move）按钮：一个新的窗口会弹出来。用鼠标缩减（或增长）分区，或者如果你喜欢，用箭头也行??use the arrows??。

    在放缩命令下达之后，点击应用（Apply），因为在你未点击应用（Apply）之前，没有任何操作施于你的硬盘之上。

    操作时间视乎分区新的的大小而定。

    操作了分区表之后请登出sidux并重启你的系统进入sidux以重读新的分区表。

NTFS分区放缩

放缩NTFS分区要求你重启系统！在重启之前不要对这个分区执行任何操作，否则你会得到错误。

Open a shell and enter the following commands:See Partitioning your HD - Disk Naming

su
apt-get update && apt-get install ntfs-3g
umount /media/xdxx
mount -t ntfs-3g /dev/disk/by-uuid/xxyyzz[etc] /media/xdxx
To get out of the konsole type: exit

An example to add ntfs to /etc/fstab after installing ntfg-3g for a permanent mount point is :

 UUID=486E97C46E97A8E8                         /media/disk3part1    ntfs         defaults  0    0

Now your NTFS Volume should be mounted rw and you should be able to store data on it. But again, be warned! Use it in emergency situations, it is not recommended for use on a daily basis.

exfat

exfat is a proprietary file system covered by patents held by Microsoft ™ and commonly found on SDXC cards. To mount/read/write to the file system, simply install the exfat-fuse package:

apt-get update && apt-get install exfat-fuse
